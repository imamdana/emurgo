## How to run


Since the base of this project is old project, these config are running well in Windows:

- Windows 11
- Node 12.22.12
- Truffle 4.0.4 


## Prerequisite

- Ganache already running at localhost:7545, refer to `truffle.js` file to modify

## Running

- npm install
- npm i -g truffle@4.0.4
- truffle compile
- truffle migrate
- open localhost:8080
